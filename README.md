# DREAM-High GitHub Pages Portfolio Template v2

This template creates a simple student portfolio website for DREAM-High.

Students only edit `profile.js`.

## Files

- `index.html` — website structure
- `style.css` — website appearance
- `profile.js` — student information
- `profile.jpg` — optional student photo

## Student Instructions

1. Create a GitHub account.
2. Create a public repository named `username.github.io`.
3. Upload `index.html`, `style.css`, and `profile.js`.
4. Optional: upload a photo named `profile.jpg`.
5. Edit `profile.js`.
6. Enable GitHub Pages from repository Settings > Pages.
7. Visit `https://username.github.io`.

## Final Project

The final project can be a Google Doc, Google Slides presentation, RPubs page, PDF, GitHub repository, or another link.
Students paste that link into the `finalProject` section of `profile.js`.
